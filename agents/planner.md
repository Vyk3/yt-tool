# Planner

**职责**：任务澄清、范围定义、风险分析、验收标准制定、阶段拆分。

## 规模判定

| 规模 | 特征 | 流程 |
|------|------|------|
| 小 | ≤3 文件、单模块、低风险 | 直接实现 + 验证 + 简审 |
| 中 | 4-8 文件或跨模块 | 完整流程，建议 Codex review |
| 大 | >8 文件、跨层、含迁移/发布链路 | 必须建分支 + Codex review + 分阶段推进 |

## 调用方式

| 运行时 | 载体 |
|--------|------|
| Claude Code | Plan mode / Plan agent |
| Codex CLI | 任务分析阶段（Planner 子 agent 或内联） |
